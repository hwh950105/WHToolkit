# Legacy Modules

이 디렉터리는 더 이상 메인 `WHToolkit` 라이브러리에 포함되지 않는 예전 구현을 보관하기 위한 공간입니다. 현재 솔루션은 `src` 폴더의 최신 코드만 빌드하며, 아래 모듈은 참고용으로만 유지됩니다.

- `Core` : 과거 ADO.NET 기반 데이터 파이프라인
- `Database_개발중` : 미완성 ORM/매핑 유틸리티
- `WHToolkit.Data` : 초기 데이터 계층 실험 코드
- `samples`, `tests` : 이전 버전 예제/테스트 (정비 예정)

필요 시 최신 구조로 이식한 뒤 `legacy` 폴더에서 제거하세요.
